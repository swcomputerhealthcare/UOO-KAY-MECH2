# Uoo-Kay-Mech Industries

Custom Precision Machined Components & Industrial Fabrication website. Built for durability, accuracy, and high-tolerance catalog specifications.

## Technology Stack
- **Framework:** Next.js (App Router)
- **Styling:** Vanilla CSS & Tailwind CSS Integration
- **Animation:** GSAP (GreenSock Animation Platform) + `@gsap/react`
- **Metrology & Compliance:** Integrated schema representation for B2B supplier verification.

## Folder Directory
- `/src/app`: Routes, sitemaps, manifest metadata, and client page modules.
- `/src/components`: Shared components including Navigation, Footer, and client-side carousels.
- `/public`: Optimized assets, blueprints, compliance badges, and logo systems.

## Development Setup
Run the development environment locally:
```bash
npm run dev
```

Build the production-optimized static bundle:
```bash
npm run build
```
