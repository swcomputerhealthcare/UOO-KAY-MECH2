11. Glossary
Term	Definition
LCP	Largest Contentful Paint – Core Web Vital measuring loading speed
CLS	Cumulative Layout Shift – measures visual stability
INP	Interaction to Next Paint – measures responsiveness
WEBP	Next-generation image format with superior compression and quality
JSON-LD	JavaScript Object Notation for Linked Data – structured data format for search engines
SSG	Static Site Generation – pages pre-rendered at build time for maximum performance
SSR	Server Side Rendering – pages rendered per request for dynamic content
MSME	Micro, Small & Medium Enterprises – government registration category
GST	Goods and Services Tax – Indian indirect tax registration
Udyam	India's online MSME registration portal and certificate
AOS	Animate on Scroll – JavaScript library for scroll-triggered animations
EmailJS	Client-side service to send emails without a backend server
CTA	Call to Action – button or link designed to prompt user interaction
OG	Open Graph – metadata protocol for social media sharing previews
CDN	Content Delivery Network – distributed server network for fast global content delivery
WCAG	Web Content Accessibility Guidelines – international accessibility standards
RFQ	Request for Quotation – formal document requesting pricing from a supplier
